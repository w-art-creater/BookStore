# 材料类图书在线销售系统

==注：此项目为本人毕业设计，仅供学习和交流使用！！！==

时间：2022年6月12日

开发环境：IDEA + Maven + JDK11。

前端技术：HTML、CSSa、JavaScript、jQuery、Ajax。

后端技术：Tomcat、Servlet、MyBatis。

数据库：MySQL。

该项目也涉及一些简单的Vue知识，搭配了Element-UI组件。

## 实现功能：

登录注册

加购图书

个人订单

图书展示

图书管理

订单管理

用户管理
